<h1 align="center">Kaiser-finance-manager</h1>

## 📖 About the project (Sobre o Projeto)

This project is a personal finance management tool built with Python and SQLite. It was developed to mark my transition from Front-end (HTML, CSS, and JavaScript) to Back-end development, focusing on data integrity and robust logic with Python.

Este projeto é um gestor de finanças pessoais construído com Python e SQLite. Ele marca a minha transição do Front-end (HTML, CSS e JavaScript) para o Back-end, focando em integridade de dados e lógica robusta com Python.

---

## 🚀 Features (Funcionalidades)

### English
* **Transaction Management:** Easy registration of both incomes and expenses.
* **Data Persistence:** Full integration with SQLite3, ensuring your financial data is safely stored.
* **SQL Integrity:** Implementation of secure queries to prevent common vulnerabilities.
* **Automated Balance Calculation:** Real-time logic to calculate your monthly balance.
* **Database Auto-provisioning:** The system automatically creates the database and tables on the first run.
* **Clean CLI Interface:** A minimalist and efficient Command Line Interface.

### Português
* **Gerenciamento de Transações:** Registro simplificado de receitas e despesas.
* **Persistência de Dados:** Integração completa com SQLite3, garantindo armazenamento seguro.
* **Integridade SQL:** Implementação de consultas seguras para evitar vulnerabilidades.
* **Cálculo Automático de Saldo:** Lógica em tempo real para calcular o saldo mensal.
* **Provisionamento Automático:** O sistema cria o banco de dados e as tabelas na primeira execução.
* **Interface CLI Limpa:** Uma Interface de Linha de Comando minimalista e eficiente.

## ⚙️ How to run (Como Rodar)

### English
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/kaiser-finance-manager.git](https://github.com/your-username/kaiser-finance-manager.git)
   ```
2. **Navigate to the folder:**
   ```bash
   cd kaiser-finance-manager
   ```
3. **Run the application:**
   ```bash
   python main.py
   ```
> **Note:** The system will automatically create the database file on its first execution.
